<h2>Telas e Funções</h2>
<h3>Sobre</h3>
<p> [ ] Texto contendo informações gerais.</p>

<h3>Conhecimentos</h3>
<p> [ ] Lista completa de cursos. Deve conter o nome do curso, uma descrição, a data de conclusão ou status do curso e um espaço para que possa acrescentar ferramentas relacionadas ou conhecimentos extras.</p>

<h3>Projetos</h3>
<p> [ ] Cartões com projetos. Cada cartão deve ter um título do projeto, uma área de mídia (1 vídeo, 2 imagens e um botão de "Abrir galeria" ou "+X") e uma breve descrição.</p>

<h3>Info. Serviços</h3>
<p> [ ] Informações sobre serviços. Contendo os serviços realizados, descrições detalhadas sobre pagamentos desses serviços, contato e formas de pagamento.</p>

<h3>Galeria</h3>
<p> [ ] Será exibida apenas ao clicar em "Abrir galeria/+X". Será uma área para exibir o projeto completo. Nome, categoria, descrição, vídeo e imagens.</p>
